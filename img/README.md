# Image assets

Drop the real files here. Nothing in this project loads images from an external CDN;
missing files degrade to a dark placeholder tile (see `MP.imageFallback()` in js/components.js).

## Site images

| File | Used by | Ratio / target size |
|------|---------|---------------------|
| `logo.png` | header, drawer, footer, favicon, OG | square, transparent PNG, ~512px |
| `hero-1.jpg` `hero-2.jpg` `hero-3.jpg` | homepage hero slider | wide ~16:9, min 1600×1000 |
| `parallax-banner.jpg` | "Design that captivates" | wide, min 2000×1200, subject centered |
| `decant-1.jpg` … `decant-4.jpg` | "Pure Decants" banner | wide, min 1900×1100 |
| `niche-1.jpg` … `niche-3.jpg` | Niche section backgrounds | wide, min 1900×1100, allow 15% bleed |
| `decanting-explainer.jpg` | "What is Decanting?" | ~1:1, min 1200×1100 |
| `gallery-1.jpg` … `gallery-8.jpg` | footer marquee + About gallery | square 1:1, min 800×800 |
